# Vision Desktop App (Python + TensorFlow)

Приложение для распознавания объектов на фотографиях и их аннотации.

Особенности:
- Поддержка обучения с учителем (обучение собственных моделей, без использования предобученных весов)
- Поддержка нескольких моделей (переключение в UI)
- Аннотация: выделение объекта на фотографии (bounding box) и сохранение метки
- Отображение параметров модели в UI
- Экспорт распознанных фото с указанием процентной уверенности

Требования:
- Python 3.9+
- TensorFlow 2.x
- PySide6
- opencv-python

Установка (Windows PowerShell):

```powershell
python -m venv venv; .\venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Запуск:
```powershell
python -m vision_desktop_app.main
```

Структура проекта:
- `annotator/` — код для аннотирования изображений
- `models/` — сохраненные модели и конфиги
- `training/` — скрипты для обучения моделей
- `ui/` — визуальные компоненты приложения
- `utils/` — вспомогательные модули

Примеры быстрого старта:

1) Установите зависимости и создайте виртуальное окружение (Windows PowerShell):
```powershell
python -m venv venv; .\venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2) Сгенерируйте тестовый датасет (для проверки):
```powershell
python -m vision_desktop_app.training.sample_data_generator
```

3) Запустите приложение (запускайте из корня проекта и в активированном virtualenv):
```powershell
# Рекомендуемый способ запуска: используйте модульный запуск `-m` из корня проекта
# Запуск GUI
python -m vision_desktop_app.main

# Примеры: тест dataset_utils (запустите в venv и из корня проекта)
python -m vision_desktop_app.training.dataset_utils --dataset data/sample_dataset --count 10 --mode detection

# Пример запуска примера (генерация датасета + короткое обучение)
python -m vision_desktop_app.examples.run_example
```

4) В UI:
 - Вкладка "Аннотация": загрузите фото, нарисуйте рамки, подпишите метки и сохраните аннотацию (`.json`).
 - Вкладка "Обучение": выберите модель (например `station_objects`), укажите папку с данными (файлы картинок и `.json` аннотации рядом), задайте параметры и запустите обучение.
 - Вкладка "Предсказание": загрузите изображение, выберите модель, нажмите "Сделать предсказание"; экспортируйте изображение с предсказанными рамками и JSON с уверенностью.
